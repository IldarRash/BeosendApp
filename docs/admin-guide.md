# Памятка администратора BeoSand

Краткое руководство по работе с админ-консолью. Описывает реальный функционал
консоли (`apps/admin`). Все расчёты (цены, места, статусы, доступность) выполняет
сервер — консоль их только показывает.

## Вход

1. Откройте консоль → страница **«Вход»**.
2. Нажмите кнопку **Telegram Login** и подтвердите вход своим аккаунтом.
3. Доступ есть только у аккаунтов из списка администраторов. Если вы не админ —
   будет ошибка 403.

> Меню слева разбито на 3 блока: **Расписание**, **Корты**, **Коммуникации**.

---

## 1. Базовая настройка (делается один раз)

В таком порядке проще всего:

1. **Уровни** — заведите уровни подготовки (новичок, средний и т.д.).
2. **Тренеры** — добавьте тренеров. Укажите Telegram ID / @username, чтобы у
   тренера открылся его интерфейс в боте.
3. **Администраторы** — при необходимости добавьте коллег. Можно добавить по
   @тегу заранее — числовой ID подставится при первом входе.
4. **Группы** — создайте регулярные слоты: дни недели, время, тренер, уровень,
   вместимость, цена за занятие и за месяц. Корт можно не указывать (назначится
   автоматически).

---

## 2. Ежемесячная рутина

1. **Тренировки → «Сгенерировать все группы»** — выберите год и месяц, консоль
   создаст тренировки на весь месяц по всем активным группам.
   - Если у группы не нашлось свободного корта, она будет помечена как
     «пропущена» — назначьте корт вручную.
2. Проверьте результат в **Тренировки** (вид «Таблица» или «Календарь»).

---

## 3. Ежедневная работа с тренировками

В разделе **Тренировки** по каждой строке доступно:

- **Записанные** — посмотреть состав.
- **Добавить человека** — записать существующего клиента или «с улицы».
- **Вместимость** — изменить число мест (нельзя меньше уже записанных).
- **Отменить** — отменить тренировку; записанные клиенты получат уведомление.

**Посещаемость** — выберите дату и тренировку, отметьте каждого:
**Пришёл / Не пришёл**.

---

## 4. Клиенты и абонементы

- **Клиенты** — поиск по имени/@тегу, регистрация новых (нужен Telegram ID +
  имя).
- **Абонементы** — месячные абонементы по группам. Кнопка **«Отметить
  оплаченным» / «Снять оплату»** меняет статус оплаты за весь месяц.

---

## 5. Корты (Edition 2)

- **Заявки на корты** — вкладка **«Ожидают»**: по каждой заявке **Подтвердить**
  (выбрать свободный корт из списка) или **Отклонить**. Клиент видит номер корта
  только после подтверждения.
- **Блокировки кортов** — вручную заблокировать корт на дату (ремонт, турнир):
  **Добавить блокировку**, при необходимости **Сменить корт** или **Удалить**.
- **Загрузка кортов** — сетка занятости по часам на выбранную дату. Внизу блок
  **«Без корта»** — назначьте корт тренировкам, у которых его нет.

> Важно: на один час нельзя подтвердить больше заявок, чем свободных кортов —
> сервер не даст.

---

## 6. Коммуникации

- **Рассылки** — выберите тип (сегодня / завтра / неделя / освободившиеся места)
  и аудиторию, проверьте **Предпросмотр** (охват + текст), нажмите
  **Отправить**.
- **Аналитика** — отчёты за период: бронирования, заполняемость, отмены, неявки,
  загрузка тренеров, эффективность рассылок.
- **Тексты** и **Шаблоны уведомлений** — редактирование надписей интерфейса и
  текстов уведомлений клиентам (с подстановкой {плейсхолдеров}).

---

## Что стоит помнить

- Все расчёты (цены, места, статусы) делает сервер — консоль их только
  показывает.
- Удаление групп/тренировок — «мягкое»: данные не пропадают, клиенты получают
  уведомления.
- Деньги — всегда в динарах (RSD).
